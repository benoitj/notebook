#+title: gentoo
* useful commands

** find out in average how long it takes to compile a package

its based on your build history

The tool is called `qlop` and part of [[https://wiki.gentoo.org/wiki/Q_applets][q applets]] package.

Example:
#+BEGIN_SRC bash
qflop -Ht qtwebengine
#+END_SRC

install with this command:
#+BEGIN_SRC bash
emerge --ask app-portage/portage-utils
#+END_SRC
