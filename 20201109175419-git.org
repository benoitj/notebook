#+title: git

* Config
** IntelliJ, cli, and Azure DevOps
for all features to work, you need to clone the https repo, but flip to ssh on CLI

snippet of doing this in .gitconfig

#+BEGIN_SRC ini :file .gitconfig :noeval
[url "git@ssh.dev.azure.com:v3/<ORG>/<PROJ>"]
    insteadOf = https://some-ads-url/<ORG>/<PROJ>/_git
#+END_SRC
