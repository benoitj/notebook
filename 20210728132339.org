#+title: sign unsigned git commits

* principle

to sign unsigned commits you need:
 1. [[id:28fbbd09-7720-4094-aa66-cf18782bc999][list signed/unsigned git commits]]
 2. rebase interactive these commits
 3. set the unsigned commits to ~edit~
 4. ~git commit --amend --no-edit~
 5. ~git rebase --continue~ until done
