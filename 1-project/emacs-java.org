#+TITLE: Migration from intellij to emacs lsp-java



* features used
** same file text search and replace

Intellij supports 2 things to do this:
 1. C-f / C-r to search within the same file (and replace if needed). supports regex
 2. multi cursor

In emacs doom, I can accomplish this in couple of ways:
 1. evil '/' search, then commands. and use 'n' to go next and '.' to repeat command
 2. evil :%s/// supports regex
 3. the /replace-regex/ and /replace-string/ function
 4. /occur/ function searches within the buffer and display an editable buffer

** all project files text search and replace

In intelliJ, this is the C-S-f and C-S-r.

Similar to same file, but accross files.

In emacs doom, I can accomplish this in couple of ways:
 1. projectile + ripgrep: search in ivy all matches
 2. in the ivy buffer, open occur using the C-c C-e button. the resulting buffer is editable
 3. /vc-git-grep/ function
