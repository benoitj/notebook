#+title: convert unicode to ascii in java

This is done in 2 step:
1. decompose unicode char into sequence of char
2. filter out non-ascii char

The code below implements this:

#+BEGIN_SRC java
String convertedString = Normalizer.normalize(input, Normalizer.form.NFD).replaceAll("[^\\p{ASCII}]", "");
#+END_SRC
